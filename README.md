# 📊 HR Analytics Dashboard | Power BI

An interactive **HR Analytics Dashboard** built using **Power BI** to analyze employee data and provide workforce insights.

## 📌 Project Overview

This dashboard helps analyze employee information using interactive visualizations and KPIs. It enables HR professionals to explore workforce trends such as employee count, salary, age, department distribution, education field, and overtime.

---

## 🎯 Objectives

- Analyze employee demographics
- Compare department-wise employee distribution
- Monitor salary and age metrics
- Explore overtime trends
- Enable interactive filtering

---

## 📊 KPIs

- 👥 Total Employees
- 💰 Total Monthly Salary
- 🎂 Average Employee Age

---

## 📈 Dashboard Features

- Department-wise Employee Analysis
- Education Field Distribution
- Gender Distribution
- Job Role Analysis
- Overtime Analysis
- Interactive Slicers

---

## 🛠 Tools Used

- Power BI Desktop
- Power Query
- DAX
- CSV Dataset

## 📊 Visualizations
